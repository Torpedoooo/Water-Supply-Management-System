This function implements the Breadth-First Search (BFS) algorithm to find an augmenting path in a graph. It starts from a given source vertex s and explores the graph until it reaches the target vertex t or until there are no more unvisited vertices reachable from the source vertex s. - 0(V + E)

Parameters
gthe provided graph
ssource node
ttarget node
Returns
Return true if a path to the target is found, false otherwise

◆ pipe_out_impact()

void Network::pipe_out_impact ( Graph< std::string >  g)

Function to calculate impact that each pipe has on each city. This function is called only once, since for each iteration (for each pipe in the graph), the cities affected by the said pipe are stored in a map, making it unnecessary to iterate through all edges for each and every city. - O( (V * E²) * E)

Parameters
gthe network's graph

void Network::testAndVisit ( std::queue< Vertex< std::string > * > &  q,

This function is used within the BFS algorithm to test whether a vertex w is not visited and if there is residual capacity available in an edge e. If both conditions are met, it marks the vertex w as visited, sets the path through which it was reached, and enqueues it for further exploration. - 0(1)

eedge to test
wvertex to visit
